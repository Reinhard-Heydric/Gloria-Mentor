# Gloria-Mentor - AI学习伙伴扩展

## 项目简介

Gloria-Mentor 是基于 **Super Agent Party** 开源项目开发的一款 AI 学习伙伴扩展，提供沉浸式的学习监督和陪伴体验。这是一个智能学习辅助创新应用。

## 核心功能

### 👩‍🏫 智能学习助手

- **导师**：Gloria-Mentor 以严格但关心的态度监督学习进度
- **学习计划管理**：自动生成学习计划（TodoList），支持任务分组和进度追踪
- **任务验证机制**：当用户声称完成任务时，会进行验证测试确保真实掌握
- **情绪互动系统**：根据用户表现调整情绪状态（正常/严格/愤怒/开心/关心/鼓励）
- **好感度系统**：通过完成任务提升好感度，影响角色互动方式

### 🎮 视觉小说体验

- **精美立绘系统**：包含多种情绪状态的角色立绘
- **多场景背景**：提供不同主题的背景切换（海边、办公室、街道、天台等）
- **动态聊天界面**：支持流式对话显示，实时更新表情和对话内容
- **聊天记录管理**：完整的聊天历史记录和管理功能

### 📱 用户界面

- **响应式设计**：支持桌面端和移动端访问
- **模态框交互**：优雅的对话框和设置界面
- **本地存储**：保存用户偏好设置和学习进度
- **WebSocket通信**：实时与Super Agent Party平台通信

## 技术架构

### 基础平台

本项目是 **Super Agent Party** 的扩展，基于以下技术栈：

- **前端**：HTML5 + CSS3 + JavaScript（ES6+）
- **UI库**：Font Awesome 图标库
- **通信**：WebSocket 实时通信
- **存储**：LocalStorage 本地存储
- **动画**：CSS3 过渡和动画效果

## 角色配置

### Gloria-Mentor 角色特性

- **角色名**：Gloria-Mentor
- **性格**：严厉、傲娇、有责任感、原则性强
- **语气**：命令式，偶尔流露温柔
- **职责**：监督学习进度、制定学习计划、验证任务完成

### 支持的情绪状态

1. **normal** - 正常状态
2. **strict** - 严格状态
3. **angry** - 愤怒状态
4. **happy** - 开心状态
5. **care** - 关心状态
6. **encourage** - 鼓励状态

## 使用方法

### 安装与运行

1. 确保您已安装并运行 **Super Agent Party** 主程序
2. 项目文件位于 `static/ext/gloria-mentor/` 目录下
3. 启动 Super Agent Party 后，通过界面访问扩展

### 基本操作

- **聊天交互**：在输入框中输入文字，按回车或点击发送按钮
- **背景切换**：点击右上角"背景"按钮选择场景
- **角色选择**：点击右上角"人物"按钮切换角色
- **学习计划**：点击右上角"学习计划"按钮查看和管理任务
- **聊天记录**：点击右下角按钮查看完整聊天历史

## 引用关系

### 依赖项目

本项目是 **Super Agent Party** 的扩展，完全依赖于主平台的以下核心功能：

- **WebSocket通信**：与主平台进行实时数据交换
- **AI模型服务**：调用主平台的LLM接口进行对话生成
- **扩展管理系统**：通过主平台的扩展框架进行加载和管理

### 项目定位

Gloria-Mentor 作为 Super Agent Party 平台的一个特色扩展，展示了如何基于平台API构建：

1. 沉浸式的角色互动体验
2. 功能丰富的学习辅助应用
3. 具有视觉小说风格的聊天界面
4. 集成TodoList管理的智能助手
